Add a Reserve

The most direct way to add reserves to a class is to click on the "Add a Reserve" tab, which gives you access to wizards that will walk you through each of the four ways you can add materials to your class in ReservesDirect. The home page of this tab displays a list of your current and future classes; select which class you would like to add reserves to. If you do not see the class you are looking for, you may create a new class or reactivate one of your previous classes using the links at the bottom of the page.

To add a reserve, click on the title of the class you would like to add the item to. You are presented with four options for adding a Reserve:

  • Search the Archive
  • Upload a Document
  • Add a URL
  • Fax a Document
