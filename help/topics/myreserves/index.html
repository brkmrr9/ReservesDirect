MyReserves

Registering for Classes | Removing Classes
Viewing Classes you are Teaching or Proxying for

The MyReserves tab displays all of the classes for which you are currently enrolled and any classes for which you are the instructor or the proxy.

Registering for Classes

To register for a class, simply click on the "Add a Class" link on the MyReserves home page. Lookup your class by instructor or by department. The search results screen will display all classes active for the current semester. Find your class and click "Add this class."

Removing Classes

To remove a class from your list of classes, just click on "Remove a Class" from the MyReserves home page. Next to the class you would like to remove, click on "Remove this class." You may not remove any classes for which you are the instructor or proxy. These will automatically disappear from your list after the class expiration date.

Viewing Classes You Are Teaching or Proxying For

The MyReserves home page automatically displays any classes that you have permission to edit that are currently active or will be active in the future. Classes you can edit display a pencil icon next to the class name. Clicking on the class name will take you to the "Edit Class" view for the class.
